# Flask简单使用小案例

综合使用html,css,javascript对网站功能界面进行改进，同时学习使用python的Flask Web框架进行网站开发。

## 使用方式

1. 安装Flask依赖
   pip install -r requirements.txt
   ![alt 安装依赖](./static/img/img1.png)

2. 运行项目
   python server.py
   ![alt 运行项目](./static/img/img2.png)

3. 打开网页界面
   在浏览器中输入提示的网址访问网页
   ![alt 网页界面](./static/img/img3.png)
